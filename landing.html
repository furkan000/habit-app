Habit Tracker - Multi-Tenant System

This is a multi-tenant habit tracking application. Each tenant has their own isolated database and data.

How It Works

To access your personal habit tracker, you need to provide a tenant name in the URL.

Your tenant name can contain:

  • Letters (a-z, A-Z)
  • Numbers (0-9)
  • Hyphens (-)
  • Underscores (_)

URL Format

http://localhost:3000/?tenant=YOUR_NAME

Examples:

  • http://localhost:3000/?tenant=alice
  • http://localhost:3000/?tenant=bob
  • http://localhost:3000/?tenant=family
  • http://localhost:3000/?tenant=work-habits

Privacy & Data Isolation

Each tenant has:

  • A separate database file
  • Complete data isolation from other tenants
  • Independent habit lists and tracking

Note: Choose a unique tenant name. Anyone with the same tenant name can access that tenant's data.

Getting Started

1. Choose a tenant name (e.g., your name, nickname, or group name)

2. Click one of the example links above or create your own URL

3. Start tracking your habits!
